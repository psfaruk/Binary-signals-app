   Binary Signals App — ইনস্টলেশন গাইড (বাংলা)
═══════════════════════════════════════════════════════════════════════

নিয়ম খুব সহজ — শুধু ২টি কাজ করতে হবে:


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  প্রথম কাজ: Python ও Git ইনস্টল করুন (শুধু একবার)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

১. Python ইনস্টল করুন:
   • https://www.python.org/downloads/ এ যান
   • "Download Python" ক্লিক করুন
   • ইনস্টল করার সময় ⚠️ "Add Python to PATH" অবশ্যই চেক করবেন
   • "Install Now" ক্লিক করুন

২. Git ইনস্টল করুন:
   • https://git-scm.com/download/win এ যান
   • "Click here to download" ক্লিক করুন
   • ইনস্টল করুন (সব default অপশনে Next ক্লিক করুন)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  দ্বিতীয় কাজ: install.bat চালান (একবারই)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

১. এই folder-এ install.bat ফাইলে double-click করুন
২. অ্যাপ নিজে থেকে সব করবে:
   ✅ App download করবে
   ✅ Dependencies ইনস্টল করবে
   ✅ start.bat চালু করবে


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  এরপর থেকে প্রতিদিন যা করবেন
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

শুধু start.bat ফাইলে double-click করুন।

বাকিটা অ্যাপ নিজে করবে:
  ✅ সার্ভার চালু হবে
  ✅ Quotex-এ auto-connect হবে (session.json বা .env থেকে)
  ✅ ব্রাউজার নিজে থেকে খুলবে (৫ সেকেন্ড পরে)
  ✅ চার্ট ও সিগন্যাল দেখাবে

প্রথমবার চালু করার সময় আপনার Quotex email + password দিতে হবে।
এরপর থেকে আর কিছু দিতে হবে না — অটো-কানেক্ট হবে।


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  সমস্যা হলে
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

সমস্যা ১: "Python ইনস্টল নেই"
  → Python ইনস্টল করুন (উপরের নিয়মে)

সমস্যা ২: "Git ইনস্টল নেই"
  → Git ইনস্টল করুন (উপরের নিয়মে)

সমস্যা ৩: Quotex login fail
  → ব্রাউজারে https://market-qx.trade এ login করে আসুন
  → তারপর আবার start.bat চালান

সমস্যা ৪: ব্রাউজার নিজে খুলল না
  → নিজে ব্রাউজার খুলে http://localhost:8000 লিখুন


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  বন্ধ করতে
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

যে Command Prompt window-এ সার্ভার চলছে সেটি বন্ধ করুন
অথবা Ctrl+C চাপুন।
